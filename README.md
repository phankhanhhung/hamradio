# HamRadio — SDR/DSP Simulation Platform

A modular ham radio simulation platform built with Java + C JNI for DSP/RF signal processing.

## Building

```bash
make all
```

## Project Structure

```
hamradio/
├── src/main/java/com/hamradio/   # Java source
├── src/main/c/                    # C native DSP/RF
├── Makefile                       # Build system
└── pom.xml                        # Maven configuration
```
